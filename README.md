# Simple ERC-20 Token

This repository contains a professional-grade implementation of an ERC-20 token. It is designed to be flat, easy to understand, and ready for deployment on any EVM-compatible blockchain (Ethereum, Polygon, BSC, etc.).

## Features
* **Standard ERC-20:** Fully compliant with EIP-20.
* **Fixed Supply:** Total supply is minted to the deployer's address at creation.
* **Security:** Uses OpenZeppelin's industry-standard libraries.

## Getting Started
1. Copy the code from `SimpleToken.sol`.
2. Open [Remix IDE](https://remix.ethereum.org/).
3. Create a new file and paste the content.
4. Compile and deploy using the Injected Provider (MetaMask).
